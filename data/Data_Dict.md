
## Hubble Data

| Column   | Units | Description                                      |
|----------|-------|--------------------------------------------------|
| CID      | None  | The candidate identifier                         |
| IDSURVEY | None  | Which survey the candidate was observed          |
| z        | None  | The observed redshift of the candidate           |
| z_ERR    | None  | The error or uncertainty in the measured z       |
| MU       | None  | The measured distance modulus                    |
| MU_ERR   | None  | The error or uncertainty in the distance modulus |
